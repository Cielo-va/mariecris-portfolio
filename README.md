# Mariecris Olorvida Portfolio Website

Premium portfolio website built with:

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons
- GitHub Pages deployment workflow

## What is included

- Home page with premium hook
- About page
- Services page
- Results page
- Customer Feedback page
- Certificates page
- Process page
- Portfolio page
- Professional Experience page
- Tools and Tech Stack page
- Contact form
- Downloadable resume
- Downloadable certificates PDF

## GitHub Pages Setup

1. Create a new GitHub repository.
2. Upload all files and folders from this project to the repository root.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, set **Source** to **GitHub Actions**.
5. Go to **Actions** and run the workflow, or simply push/commit changes to the `main` branch.
6. Once the workflow is green, GitHub will provide your live website link.
